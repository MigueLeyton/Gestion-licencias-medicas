Para poder ejecutar el proyecto lo primero que deben hacer es instalar todo lo que viene en la configuracion del package.json, para esto ejecutan un npm install en la terminal y listo.

Este programa es lo que se conoce como API REST, con esto podremos tener el frontend separado del backend, y como este ultimo es independiente, no importa el frontend que hagamos, siempre podremos comunicarnos con el backend.

Para poder entender el programa les recomiendo seguir este orden

index.js -> database -> models -> controllers -> middlewares ->routes

Que hace cada uno:

- En el index.js definimos la configuración de nuestro servidor, cargamos el .env, definimos algunas cosas como el puerto o algunas configuraciones
- En database realizamos la configuración de la base de datos, obteniendo los datos de configuración desde el archivo .env
- En models hacemos todo lo que esté relacionado con la base de datos, consultas, insertar nuevos datos, etc. Aquí no se hace validaciones de los datos que vienen en la petición, eso lo hace previamente el controlador.
- En controllers recibimos los datos de las peticiones, acá podemos hacer lo que queramos con ellos, modificarlos, validarlos, o lo que tengan que hacer con ellos.
- En routes definimos los endpoints o los puntos a los cuales nos comunicaremos desde el frontend.
- En middlewares haremos todo lo que tenga que ver con validaciones que se ejecutaran antes de lo que este en controller, como por ejemplo verificar algún token, verificar sesión o lo que ustedes vean necesario.

Para ejecutar el programa primero importen el archivo proyecto_base.sql en el xampp, y luego ejecuten el servidor colocando " npm run dev " en la terminal (La terminal debe estar ubicada en esta carpeta)
